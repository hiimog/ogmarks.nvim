# ogmarks.nvim

Opinionated and persistent marking

## Install

`:CocInstall ogmarks.nvim`

## Keymaps

`nmap <silent> <C-l> <Plug>(coc-ogmarks.nvim-keymap)`

## Lists

`:CocList demo_list`

# Building
- make sure that `sourcemap: 'inline'` is set in `esbuild.js`
